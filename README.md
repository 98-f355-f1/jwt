# JWT

An implementation of [JSON Web Token (JWT)](https://tools.ietf.org/html/rfc7519) in
Crystal programming language.


_The project is under development_

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  jwt:
    github: greyblake/jwt
```


## Usage

```crystal
```

## Tests

```
crystal spec
```
